Projects

    • Autism Connectome Analysis [Ongoing]

      • In this project I am looking at the resting state fMRI brain scan of autistic people to develop methods to diagnose if the person has Autism, based on just this scan. Github Link

    • Face tracking using quad-rotor drone

      • The aim of this project was to make the drone to track the human face using its camera. We programed it to change its yaw according to the tracked face. The drone turns to centre itself with the detected face. It is programmed to track the face of the person closest to it. The project was made on Robot Operating System (ROS) using the Autonomy package. It was implemented on Parrot AR Drone quadcopter at Robotics Lab of RGUKT, R.K. Valley, Andhra Pradesh.       Video Link

    • Master of Technology (M.Tech) Thesis

      • Title: Static hand gesture recognition using Deep Learning.
      Supervisor: Prof. G.C. Nandi
      Description: The aim of this project was to control the movements of a differential wheeled robot by using just hand gestures. It involved the use of Microsoft Kinect for tracking the gesture and a Deep Learning technique based on the combination of Denoising and Sparse Autoencoders for feature extraction and gesture recognition. The work was implemented on Social Mobile Autonomous Robot Test-bed (SMART) developed in Robotics and Artificial Intelligence Lab of IIIT-Allahabad.       Video Link

    • Mini Project

      • Title: Hand written digits recognition
      Supervisor: Prof. G.C. Nandi
      Description: The aim of this project was to use stacked Sparse Autoencoders to extract features from the image of a digit and classify it as a number from 0 to 9. The architecture was trained and tested on MNIST dataset.

    • Bachelor of Technology(B.Tech) Project

      • Title: Barcode enabled student registration and course management system.
      Supervisor: Mr. Pankaj Tiwari
      Description: The aim of this project was to completely automate the student registration system of college. It also automated the task of course selection of students thereby completely eliminating the paper-work. It was a windows based project based on 3-Tier Architecture.
